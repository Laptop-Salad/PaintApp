# Paint App

This is a simple paint app made using python. It was made using tkinter, PILs Image and ImageDraw libraries. 


![image](https://user-images.githubusercontent.com/80591698/216789217-0a202f6a-13fe-44f2-a9a5-7b3089b4513d.png)
<br>
The paint app starts out with a blank canvas.

<br>

![image](https://user-images.githubusercontent.com/80591698/216789253-95b9ce24-d04d-4af9-ae89-13bfcb7d6976.png)
<br>
To start drawing left click and move your mouse around the canvas to draw, the default colour is black and you can select more below the canvas.

<br>

![image](https://user-images.githubusercontent.com/80591698/216789267-2a8205b3-b166-426b-bc4a-df2a19e58529.png)
<br>
You can also change the pen size below the colour selection by pressing the plus and minus icons. 

<br>

![image](https://user-images.githubusercontent.com/80591698/216789281-ea5e69be-4e2f-4656-9eac-bb04f57ca3ad.png)
<br>
You can then save your image as a png.

<br>

![image](https://user-images.githubusercontent.com/80591698/216789397-a437a95e-d7cc-4cca-bb89-12174b17e316.png)
<br>
Final png image.
